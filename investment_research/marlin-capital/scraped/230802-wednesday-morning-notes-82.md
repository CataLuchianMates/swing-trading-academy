# Wednesday Morning Notes 8.2

Technical Update

Tuesday was another quiet day for the indices with the SPX finishing -0.27%.

[![](https://substackcdn.com/image/fetch/$s_!LnRl!,w_1456,c_limit,f_auto,q_auto:good,fl_progressive:steep/https%3A%2F%2Fsubstack-post-media.s3.amazonaws.com%2Fpublic%2Fimages%2F0270af72-cdde-4c69-a7a4-13db2830a5f6_964x727.jpeg)](https://substackcdn.com/image/fetch/$s_!LnRl!,f_auto,q_auto:good,fl_progressive:steep/https%3A%2F%2Fsubstack-post-media.s3.amazonaws.com%2Fpublic%2Fimages%2F0270af72-cdde-4c69-a7a4-13db2830a5f6_964x727.jpeg)

We got some news post-market with Fitch downgrading the US credit rating to AA+ from AAA. The last time this happened was August 2011, which preceded a -19% drop in the S&P 500.

We are set for a gap down this morning on this news. The market has shaken off any and all negative news in 2023, so only time will tell if this catalyst will actually instill some fear in the market.

I’ll be watching to see if key downside support levels hold in the SPX, starting with the 8 EMA at $4560. Other key support levels include $4530 and $4500-$4510.

CLS

Set for a gap down this morning (along with the market), also on news of a secondary offering.

[![](https://substackcdn.com/image/fetch/$s_!5797!,w_1456,c_limit,f_auto,q_auto:good,fl_progressive:steep/https%3A%2F%2Fsubstack-post-media.s3.amazonaws.com%2Fpublic%2Fimages%2Fb2759966-7663-4f60-a578-10a100047791_973x717.jpeg)](https://substackcdn.com/image/fetch/$s_!5797!,f_auto,q_auto:good,fl_progressive:steep/https%3A%2F%2Fsubstack-post-media.s3.amazonaws.com%2Fpublic%2Fimages%2Fb2759966-7663-4f60-a578-10a100047791_973x717.jpeg)

Sometimes we can see secondary offerings kill strong momentum, so I am less interested after this. With that said, I still like the setup, volume patterns, and unusual options activity, so I will be initiating a starter position in the $20 range.
