# Tuesday Morning Notes 1.31

Technical Analysis

The market pulled back to start off the week yesterday, finishing -1.3% for the session.

[![](https://substackcdn.com/image/fetch/$s_!V4zx!,w_1456,c_limit,f_auto,q_auto:good,fl_progressive:steep/https%3A%2F%2Fsubstack-post-media.s3.amazonaws.com%2Fpublic%2Fimages%2F03f891ff-a1df-4e4f-9ba3-bdb7b821e0c0_959x721.jpeg)](https://substackcdn.com/image/fetch/$s_!V4zx!,f_auto,q_auto:good,fl_progressive:steep/https%3A%2F%2Fsubstack-post-media.s3.amazonaws.com%2Fpublic%2Fimages%2F03f891ff-a1df-4e4f-9ba3-bdb7b821e0c0_959x721.jpeg)

So far, this is a healthy and ordinary pullback after such a large rally off lows. If the pullback continues today, key support levels to watch below include $4020, $4000, and the key moving averages ($3963, $3957, and $3924).

Resistance wise, $4100 remains the key level to watch above.  If we can break above this level, the market can start to make a run towards August highs ($4300).
