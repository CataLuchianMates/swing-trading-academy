# Tuesday Morning Notes 11.1

Technical Analysis

Equities pulled back a bit to start the week. Importantly, the QQQ held the $275-$276 support level on the downside (21 EMA).

[![](https://substackcdn.com/image/fetch/$s_!YReS!,w_1456,c_limit,f_auto,q_auto:good,fl_progressive:steep/https%3A%2F%2Fbucketeer-e05bbc84-baa3-437e-9518-adb32be77984.s3.amazonaws.com%2Fpublic%2Fimages%2Ff2f98a2d-a573-445b-b482-6f15fb80bccc_1198x903.jpeg)](https://substackcdn.com/image/fetch/$s_!YReS!,f_auto,q_auto:good,fl_progressive:steep/https%3A%2F%2Fbucketeer-e05bbc84-baa3-437e-9518-adb32be77984.s3.amazonaws.com%2Fpublic%2Fimages%2Ff2f98a2d-a573-445b-b482-6f15fb80bccc_1198x903.jpeg)

Now, we are set for a sizable gap up this morning. We remain in a choppy range between the 21 and 60 EMAs.

Key resistance levels ahead include $284-$286.

Key support levels below include $275-$276.

Keep in mind the FOMC decision will be announced tomorrow.
