# Wednesday Morning Notes 8.39

Technical Notes

A big day for the SPX yesterday as we broke above and reclaimed all key moving averages.  Important to note is that we never got a bearish EMA crossover on this pullback, although we did temporarily break below my key moving averages.  The break below the 60 EMA was short lived though, as we rallied back above these key levels in relatively short order (and importantly, before the bearish EMA crossover occurred).

CLS

Great action on the recent market pullback and strong volume patterns on the daily chart. I like this stock to hit $30 on this run up (market willing).

Bullish cup and handle breakout forming. Yesterday was an inside day. Long setup above $17.02 with a stop below $16.53 (-2.9% risk).
