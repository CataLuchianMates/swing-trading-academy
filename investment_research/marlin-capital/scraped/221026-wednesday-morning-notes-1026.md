# Wednesday Morning Notes 10.26

Technical Analysis

The market followed through to the upside yesterday, breaking some key support levels in the process. The upside move was immediately reversed in after hours following Microsoft and Google results. Remember, earnings will only get (much) worse from here.

[![](https://substackcdn.com/image/fetch/$s_!0Reg!,w_1456,c_limit,f_auto,q_auto:good,fl_progressive:steep/https%3A%2F%2Fbucketeer-e05bbc84-baa3-437e-9518-adb32be77984.s3.amazonaws.com%2Fpublic%2Fimages%2F39ee5b65-dbeb-4a6e-bff9-294a53c11f90_1002x757.jpeg)](https://substackcdn.com/image/fetch/$s_!0Reg!,f_auto,q_auto:good,fl_progressive:steep/https%3A%2F%2Fbucketeer-e05bbc84-baa3-437e-9518-adb32be77984.s3.amazonaws.com%2Fpublic%2Fimages%2F39ee5b65-dbeb-4a6e-bff9-294a53c11f90_1002x757.jpeg)

We are set for a sizable gap down this morning below $280. The first key support level to watch is $278, followed by the critical $275 level (21 EMA). If $275 support can hold, it is still possible to make a higher low and for this rally to continue.

Also keep in mind Apple and Amazon are set to report earnings this week, so we have several more potential catalysts to get through.
